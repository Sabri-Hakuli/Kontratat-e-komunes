How to install in local environment

In terminal write the following commands:

***
+    git clone https://github.com/Sabri-Hakuli/Kontratat-e-komunes.git 
***
+    cd Kontratat-e-Komunes 
***
+    npm install 
***

+    mongod 
***
+    node app.js (or nodemon) 
***
>    http://localhost:3000/
